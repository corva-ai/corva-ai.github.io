Postman Tutorial

How to get started with Postman and Corva:

First download the Postman app:

https://www.postman.com/downloads/

Second Step:

Request your companies API KEY.

Open Postman app:

Step 1: Add a new environment, click the manage environments button

In the add manage environments menu > click add

Add a new environment name. (example CORVA_API_KEY)

VARIABLE: api_key

Note: be sure to include API before the api key.

INITIAL VALUE: API user_api_key

CURRENT VALUE: user_api_key

Click Add:

Now change your environment to the newly created environment.

Now you are almost good to go! Lets run through a quick use case example of Postman

click the "+" button to add a new query:

Get a query of your choice. Please look into these code snippets of FAQ queries:

(LINK TO CODE SNIPPETS)

Example:

https://api.corva.ai/v1/data/corva/wits.summary-30m?asset_id=MYASSETIDNUMBER&limit=10

Next, you have one final authorization step.

Click the "Headers" tab

add in Key "Authorization" and VALUE: {{api_key}}

This will get the api_key variable from your CORVA_API_KEY environment that you have selected

Now you are good to go! click Send

You should see some Data if the asset id has any!
