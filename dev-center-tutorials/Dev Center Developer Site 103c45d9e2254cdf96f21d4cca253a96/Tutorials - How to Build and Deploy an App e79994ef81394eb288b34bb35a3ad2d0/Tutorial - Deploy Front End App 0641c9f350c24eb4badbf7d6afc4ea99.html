Overview

This guide will show you how to:

  • Create a front end app
  • Copy an existing pre-built app
  • Connect to an existing dataset
  • Deploy an app to the Dev Center
  • Add an updated app version to the Dev Center
  • Assign an app to an asset

Prerequisites

  • Account access to Corva Dev Center: michael.cortez@corva.ai
  • Corva GitHub repository access: https://github.com/corva-ai

  • Repository of your choice e.g. GitHub, Bitbucket, etc.
  • Tech stack knowledge of React Hooks and JavaScript
  • Understand basic Command Line Interface (CLI) commands for Linux or Windows
  • Integrated Development Environment (IDE) of your choice e.g. Visual Studio Code, PyCharm, etc.
  • Download Git here: https://git-scm.com/downloads
  • Install Yarn
npm install --global yarn

Clone Pre-Built Front End App (If Applicable)

  • Choose a directory to store the app.
  • Copy the following repository link: https://github.com/corva-ai/corva-example-ui-apps.git
  • Input the following into the command line:
git clone https://github.com/corva-ai/corva-example-ui-apps.git
  • You should have the following directory for the corva-example-ui-apps.

Creating an App in Dev Center

  • Log into: https://app.corva.ai/login
  • Navigate to the ribbon and click the </> button to access Dev Center.
  • Click + NEW APP button
  • Click Frontend radio button
  • Click NEXT button
  • Populate App Name field
  • Click Drilling radio button
  • Click the applicable Category radio button
  • Click NEXT button

Creating App in IDE or Terminal

  • Within Dev Center / New App modal
  • Within Install the Command Line Tool section code snippet box, click on document icon to copy: npm i -g @corva/create-app
  • Open terminal
  • In command line paste: 
npm i -g @corva/create-app
  • Press enter
  • Go back to Dev Center / New App modal
  • Within the Run and follow CLI instructions section code snippet box, click on document icon to copy: 
create-corva-app 20210225_new_app --appName "20210225 New App" --segment "drilling" --category "asset" --appKey "dctest.20210225_new_app.stream" --appType "ui"
  • Open terminal
  • In command line paste: 
create-corva-app 20210225_new_app --appName "20210225 New App" --segment "drilling" --category "asset" --appKey "dctest.20210225_new_app.stream" --appType "ui"
  • In the command line, edit the command line snippet - - segment to - - segments and add - -runtime "ui". See revised line below.
create-corva-app 20210225_new_app --appName "20210225 New App" --segments "drilling" --category "asset" --appKey "dctest.20210225_new_app.stream" --appType "ui" --runtime "ui"
  • Press enter
  • Go back to Dev Center / New App modal
  • Click on ADD NEW APP button

Copy Existing App into Newly Created App (If Applicable)

  • Open Finder in Mac or Explorer in Windows
  • Open directory of existing Corva app e.g. corva-example-ui-apps
  • Drill down to app name e.g. sample_gamma_depth
  • Copy the following file:
    • src
  • Open directory of newly created app e.g. 20210225_new_app
  • Paste and replace the copied file

Link App to Dataset

  • Open newly created app directory e.g. 20210224gammav5
  • Open the apiCalls.js file
  • Replace the const collectionName = 'actual_gamma_app' with desired collection name. Save file. See code snippet. 
//Before updating collection name in apiCalls.js
const collectionName = 'actual-gamma-depth';

//After updating collection name in apiCall.js
const collectionName = '20210224gammav4';
  • Open the subscription.js file
  • Replace the collection: 'actual-gamma-depth' with desired collection name. Save file. See code snippet.
// Before updating the collection name
export const getGammaSubscription = (assetId, provider) => {
  return {
    provider,
    assetId,
    behavior: 'accumulate',
    collection: 'actual-gamma-depth',
    params: { limit: 10000 },
  };
};

// After updating the collection name
export const getGammaSubscription = (assetId, provider) => {
  return {
    provider,
    assetId,
    behavior: 'accumulate',
    collection: '20210224gammav4',
    params: { limit: 10000 },
  };
};
// In order to see your front end react app in http://localhost:8080/app run: 
yarn start

Upload and Publish App to Dev Center

  • In the command line type the following
yarn zip
  • Open Dev Center / Apps / New App Name / Versions
  • Drag and drop or Upload the newly zipped file
  • Type in version notes (optional)
  • Click UPLOAD button
  • Allow time for app to be queued, processed, build and deploy
  • Once in draft status, click on PUBLISH button

Add App to an Asset

  • Navigate to the Corva main page via top left Corva icon in navbar
  • Click on Assets link in navbar
  • Select and click on the Asset name or Search for desired Asset
  • Within the Asset Dashboard scroll down to bottom of page
  • Click on the +ADD APP button
  • Within App Store, search or filter to locate the newly created app
  • Click the + ADD button on the newly created app card
  • Newly created app should now appear in the Asset Dashboard
