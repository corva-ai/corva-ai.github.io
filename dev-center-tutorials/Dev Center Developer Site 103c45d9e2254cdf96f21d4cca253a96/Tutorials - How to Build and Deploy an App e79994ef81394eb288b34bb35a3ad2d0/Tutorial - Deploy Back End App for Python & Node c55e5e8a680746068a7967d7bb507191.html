Overview

This guide will show you how to:

  • Create a back end app
  • Create a new dataset
  • Copy an existing pre-built app
  • Connect to an existing dataset
  • Deploy an app to the Dev Center
  • Add an updated app version to the Dev Center
  • Assign an app to an asset

Prerequisites

  • Account access to Corva Dev Center: michael.cortez@corva.ai
  • Corva GitHub repository access: https://github.com/corva-ai
  • Repository of your choice e.g. GitHub, Bitbucket, etc.
  • Tech stack knowledge of Node.js, React, Python and Javascript
  • Understand basic Command Line Interface (CLI) commands for Linux or Windows
  • Integrated Development Environment (IDE) of your choice e.g. Visual Studio Code, PyCharm, etc.
  • Download Git here: https://git-scm.com/downloads
  • Install Node.js here: https://nodejs.org/en/download/

Clone Pre-Built Back End App (If Applicable)

  • Choose a directory to store the app.
  • For Python, copy the following repository link: https://github.com/corva-ai/dev-center-gamma-depth-python.git
  • For Node, copy the following repository link: https://github.com/corva-ai/dev-center-gamma-depth-node.git
  • Input the following into the command line:
git clone https://github.com/corva-ai/dev-center-gamma-depth-python.git
  • You should have the following directory for the dev-center-gamma-depth app.

Creating an App in Dev Center

  • Log into: https://app.corva.ai/login
  • Navigate to the ribbon and click the </> button to access Dev Center.
  • Click + NEW APP button
  • Click Backend radio button
  • Click NEXT button
  • Click stream radio button
  • Click NEXT button
  • Populate App Name field
  • Click Drilling radio button
  • Choose EnhancedWITS from dropdown menu
  • Click Analytics radio button
  • Click NEXT button

Creating App in IDE or Terminal

  • Within Dev Center / New App modal
  • Within Install the Command Line Tool section code snippet box, click on document icon to copy: npm i -g @corva/create-app
  • Open terminal
  • In command line paste: 
npm i -g @corva/create-app
  • Press enter
  • Go back to Dev Center / New App modal
  • Within the Run and follow CLI instructions section, click Python radio button
  • Within the Run and follow CLI instructions section code snippet box, click on document icon to copy: 
create-corva-app 20210216_new_app --appName "20210216 New App" --segment "drilling" --category "analytics" --appKey "big-data-energy.20210216_new_app.stream" --appType "stream" --runtime "python3.8"
  • Open terminal
  • In command line paste: 
create-corva-app 20210216_new_app --appName "20210216 New App" --segment "drilling" --category "analytics" --appKey "big-data-energy.20210216_new_app.stream" --appType "stream" --runtime "python3.8"
  • In the command line, edit the command line snippet - - segment to - - segments. See revised line below.
create-corva-app 20210216_new_app --appName "20210216 New App" --segments "drilling" --category "analytics" --appKey "big-data-energy.20210216_new_app.stream" --appType "stream" --runtime "python3.8"
  • Press enter
  • Go back to Dev Center / New App modal
  • Click on ADD NEW APP button

Create a New Data Set (If Applicable)

  • Within the Dev Center - New App view, click on Datasets Explorer on top navbar
  • Click on Dataset Type - Time-Based or Depth-Based or Reference
  • Click on + NEW DATASET button in top right corner
  • Populate values for "name": "", "description": "" and "type": "" only. Do not populate "indexes" or "schema". See dataset template below. 
//Dataset template
{
	"name": "big-data-energy#my-new-dataset",
	"description": "my new data set",
	"type": "time",
	"indexes": [],
	"schema": {}
}
  • Click on SAVE button

Copy Existing App into Newly Created App (If Applicable)

  • Open Finder in Mac or Explorer in Windows
  • Open directory of existing Corva app e.g. dev-center-gamma-depth-python
  • Drill down to app type e.g. stream or scheduled
  • Copy the following files:
    • src
    • lambda_py
    • requirements.txt
  • Open directory of newly created app e.g. 20210218_new_app
  • Paste and replace the three copied files

Upload and Publish App to Dev Center

  • Open Finder in Mac or Explorer in Windows
  • Open directory of existing Corva app e.g. 20210218_new_app
  • Select and zip up the following files:
    • lambda_py
    • manifest.json
    • requirements.txt
    • src
  • Open Dev Center / Apps / New App Name / Versions
  • Drag and drop or Upload the newly zipped file
  • Type in version notes (optional)
  • Click UPLOAD button
  • Allow time for app to be queued, processed, build and deploy
  • Once in draft status, click on PUBLISH button

Add App to an Asset

  • Navigate to the Corva main page via top left Corva icon in navbar
  • Click on Assets link in navbar
  • Select and click on the Asset name or Search for desired Asset
  • Click on Data & Settings icon in navbar
  • Within the General Info page, click on the kabob menu at top right
  • Click on Well Streams from drop down menu
  • Select Time or Depth
  • Click on Well Stream Asset name
  • Within the Streams / App Stream Asset Name page, click on + button
  • Within App Store, search or filter to locate the newly created app
  • Click the + ADD button on the newly created app card
  • Newly created app should now appear in Installed Apps
